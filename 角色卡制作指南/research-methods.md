# 如何自主查阅角色卡

## 信息来源优先级

| 优先级 | 来源 | 网址 | 特点 |
|--------|------|------|------|
| ⭐⭐⭐ | PRTS wiki | https://prts.wiki/w/<角色名> | 最权威，含语音、模组、档案原文 |
| ⭐⭐⭐ | 萌娘百科 | https://zh.moegirl.org.cn/<角色名> | 最全面，含轶事、考据、社区梗 |
| ⭐⭐ | GitHub lore wiki | https://github.com/littlepangding/arknights_lore_wiki | LLM总结版，快速了解概要 |
| ⭐⭐ | B站视频 | https://bilibili.com | 角色PV、立绘展示、剧情解读 |
| ⭐ | NGA/贴吧 | - | 玩家讨论、二创、梗 |

## 查阅流程

### 第一步：确定角色基本信息
1. 去 PRTS wiki 搜角色名
2. 提取：身高、生日、种族、职业、出身地
3. 查看干员档案（基础档案、临床诊断、客观履历）

### 第二步：获取详细设定
1. 去萌娘百科搜角色名
2. 提取：萌点、相关人士、角色经历、轶事、考据
3. 注意：萌娘百科内容由社区维护，可能有错误

### 第三步：获取语音台词
1. PRTS wiki → 语音记录
2. 提取所有中文语音（任命助理、交谈、触摸、信赖等）
3. 注意：台词必须官方来源，不能编造

### 第四步：获取立绘信息
1. PRTS wiki → 干员信息 → 立绘图片
2. B站搜索角色名 + "皮肤" 或 "立绘"
3. 使用 MiMo Vision API 分析立绘细节

### 第五步：获取剧情信息
1. PRTS wiki → 相关活动剧情
2. GitHub lore wiki → 快速概要
3. 提取角色在剧情中的关键表现

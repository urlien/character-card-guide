**鼓¯꒳¯**: 首先先介绍一下我们的目的：

想象一下，你有一个产品列表、文章合集，或任何需要分类展示的数据。

我们的目的就是为每个数据项（按钮/卡片）贴上“隐形标签”，然后通过点击按钮和输入文字，动态地告诉浏览器哪些卡片需要显示，哪些需要隐藏。

核心原理：3 个部分搭出筛选功能

就像搭积木，这个功能由 3 部分组成：

HTML 骨架：搭建筛选按钮、搜索框和要筛选的卡片（相当于搭出 “架子”）；
CSS 样式：给骨架穿 “衣服”，让按钮、卡片变好看；
简单交互：点击按钮或输入文字时，只显示符合条件的卡片（相当于给积木加 “动作”）。

**鼓¯꒳¯**: 为了方便理解，这里就以我的卡《众神之巅！》中的代码为例：

<:emoji_118:1379226234168414289> 第一步：先看 HTML 骨架（搭架子）

我们需要 3 类 “零件”：

1. 搜索框
用来输入文字搜索卡片，比如输入 “剑” 就能找到带 “剑” 字的卡片。
```html
<!-- 搜索框区域 -->
<div class="p">
  <div class="r">:mag:</div> <!-- 搜索图标 -->
  <input type="text" class="q" placeholder="搜索神魔..." 
         oninput="这里是搜索时的动作"/> <!-- 输入框 -->
</div>
```
2. 筛选按钮组
比如按 “类型”（神明 / 邪魔）和 “等级”（一等 / 二等...）筛选，点击按钮就只显示符合条件的卡片。
```html
<!-- 筛选按钮区域 -->
<div class="s">
  <!-- 类型筛选按钮：全部/神明/邪魔 -->
  <div class="t">
    <div class="u k">全部</div>
    <div class="u">神明</div>
    <div class="u">邪魔</div>
  </div>
  <!-- 等级筛选按钮：所有等级/一等/二等... -->
  <div class="t">
    <div class="u k" data-rank="all">所有等级</div>
    <div class="u" data-rank="1">一等</div>
    <div class="u" data-rank="2">二等</div>
    <!-- ...更多等级按钮 -->
  </div>
</div>
```
3. 可点击的卡片
每个卡片都藏着 “秘密信息”（比如类型、等级、名称、描述），这些信息会被筛选功能用到。
```html
<!-- 可筛选的卡片（代码3示例） -->
<div class="w" 
     data-type="god"  <!-- 卡片类型：神明 -->
     data-rank="1"   <!-- 卡片等级：一等 -->
     data-name="一等神：剑一"  <!-- 卡片名称（用于搜索） -->
     data-desc="...剑仙...">  <!-- 卡片描述（用于搜索） -->
  <div class="x">仙</div> <!-- 卡片图标 -->
  <div class="y">剑一</div> <!-- 卡片名称 -->
  <div class="aa ab">神明</div> <!-- 类型标签 -->
  <div class="ad">一等</div> <!-- 等级标签 -->
  <div class="z">...描述...</div> <!-- 详细描述 -->
</div>
```
小知识：data-xxx 是卡片的 “秘密标签”，比如 data-type="god" 表示这是 “神明” 类型，筛选时会用这些标签判断是否显示卡片。

**鼓¯꒳¯**: <:emoji_118:1379226234168414289> 第二步：用 CSS 给骨架穿衣服（变好看）

**鼓¯꒳¯**: <:emoji_118:1379226234168414289> 第三步：交互逻辑（让筛选动起来）

当你点击筛选按钮或在搜索框输入文字时，背后的逻辑很简单：

1. 先判断你选择的条件（比如 “神明”+“一等”）；
2. 检查所有卡片的 “秘密标签”（data-type/data-rank/data-name等）；
3. 只显示符合条件的卡片，隐藏不符合的。

比如搜索框的逻辑（简化版）：
```html
// 当输入文字时
input.oninput = function() {
  const searchWord = this.value.toLowerCase(); // 输入的文字（转小写，忽略大小写）
  // 遍历所有卡片
  document.querySelectorAll('.w').forEach(card => {
    const cardName = card.getAttribute('data-name').toLowerCase(); // 卡片名称
    const cardDesc = card.getAttribute('data-desc').toLowerCase(); // 卡片描述
    // 如果名称或描述包含搜索词，就显示；否则隐藏
    if (cardName.includes(searchWord) || cardDesc.includes(searchWord)) {
      card.style.display = 'block';
    } else {
      card.style.display = 'none';
    }
  });
};
```

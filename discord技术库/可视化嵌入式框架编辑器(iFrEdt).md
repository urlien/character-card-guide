# 项目介绍

该项目是为了让不懂代码的人可以快速编辑出属于自己的美观网页 甚至沉浸式游戏环境而设计的

该项目实现了非常丰富的内容 包括普通的高可定制化对话框 以及内置的一个手机模拟对话框(内置了15款应用 你也可以发布自己的应用)

详细情况可以见图

# 一切的开始

1. 你只需要添加一条正则

 `/\[([A-Z_0-9]+:[^\]]+)\]/`

2. 以及一条[通用框架注入代码](https://raw.githubusercontent.com/Nixdorfer/mmd-iframe-template/refs/heads/main/inject.html)

3. 然后新建一个正则 随便写一点什么 比如我写的是`【茶壶说：要有光】`替换内容写`[EDIT:MODAL:{{char}};;{{user}};;]`

4. 将这条正则放在刚才的框架注入代码前面

5. 在第一句话把你刚才的匹配词写进去

6. 开始你在iFrEdt的旅程!

# 项目结束

1. 点击左侧的下载 或者复制 你可以获得如下形式的编译代码

```MODAL
[MODAL_2:{{char}};;{{user}};;eJyrVipITE8tVrKKrlYqySzJSVWyUnq6fueL9XOfLWwzVKqNrQUA7ZQO2g==]
```

2. 将其粘贴到你的第一条正则 替换掉刚才的`[EDIT:MODAL_2:{{char}};;{{user}};;]`

3. 刷新项目

4. 和别人分享你的创作吧!

# 继续编辑项目

你只要在代码最开始加入EDIT:就可以附加编辑界面进去 继续编辑一个已有项目了 例如

```MODAL
[MODAL_2:{{char}};;{{user}};;eJyrVipITE8tVrKKrlYqySzJSVWyUnq6fueL9XOfLWwzVKqNrQUA7ZQO2g==]
```

是正常启动用户界面

```MODAL
[EDIT:MODAL_2:{{char}};;{{user}};;eJyrVipITE8tVrKKrlYqySzJSVWyUnq6fueL9XOfLWwzVKqNrQUA7ZQO2g==]
```

则是附加编辑器界面启动项目

# TIPS

- 第一句话里不要忘记写你给AI的回复！如果你开启了沉浸式渲染 用户则不会看到这条消息 所以请尽情发挥。不过如果你忘写了 AI则只能对着空的开场白发呆！

- 虽然你直接把iFrEdt导出的编译代码放到第一句话是可以的 但是这会占用宝贵的第一句话空间 同时AI还会收到一大堆莫名其妙的代码 因此 **你虽然可以 但最好不要这么做**

- 写着iFrEdt编译代码的正则一定要位于通用框架注入代码正则的前面！否则无法被正确捕获！

- 是的 通用框架注入代码你不需要更改 无论你是否要使用编辑器 或者编译出什么样的代码

- 我建议你在`设置正则-浏览效果`的按钮中进行编辑 否则一旦你保存了 但忘了设置隐藏按钮 就会显得有点蠢

- 如果你真的写了超长的代码 超出了一条正则的上限 那你可以把它截成两段 写进两条正则里 例如

  - 你将拥有x条正则
    1. iFrEdt代码第一部分
    2. iFrEdt代码第二部分
    3. iFrEdt代码第x部分
    4. 通用框架注入代码

  - 每条正则都有不同的匹配词 如果
    1. 第一条的匹配词为`【茶壶说：要有光】`
    2. 第二条的匹配词为`【茶壶说：要有茶】`
    3. 第三条的匹配词为`【茶壶说：要有壶】`
    4. 那么你的第一句话写`【茶壶说：要有光】【茶壶说：要有茶】【茶壶说：要有壶】` **不要换行** 就可以了

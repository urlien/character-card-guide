# 📦 侧边栏“搬运工”通用教程

本教程教你如何使用一段通用代码，将你的作品（按钮、图片、面板等）直接“搬运”到网站自带的 **左侧** 或 **右侧** 空白容器中。

### 核心功能

  * **即插即用**：复制粘贴到代码末尾即可。
  * **纯粹搬运**：只负责改变元素的位置结构。

-----

### 第一步：确认你的“货物”是谁

你需要知道你想移动的那段代码的 `class`（类名）或者 `id` 叫什么。

  * **例子**：如果你的代码最外层是 `<div class="类名/id在此处如：xxx">...</div>`
  * **你的货物名就是**：`.xxx` （记得加前面的点）

-----

### 第二步：选择“搬运”代码（二选一）

请将以下代码直接复制并粘贴到你所有代码的**最末尾**。

### 👉 选项 A：搬运到【右侧】容器 (Right Side)

```html
<img src="x" style="display:none" onerror="
    /* 1. 设定目的地（右侧容器） */
    var target = document.querySelector('.mm-right-side-container'); 
    /* 2. 设定要搬运的货物（修改下方引号里的内容） */
    var cargo = document.querySelector('.这里填你的类名'); 
    /* 3. 执行搬运 */
    if(target && cargo){ 
        target.appendChild(cargo);}">
```

### 👈 选项 B：搬运到【左侧】容器 (Left Side)

```html
<img src="x" style="display:none" onerror="
    /* 1. 设定目的地（左侧容器） */
    var target = document.querySelector('.mm-left-side-container'); 
    /* 2. 设定要搬运的货物（修改下方引号里的内容） */
    var cargo = document.querySelector('.这里填你的类名'); 
    /* 3. 执行搬运 */
    if(target && cargo){ 
        target.appendChild(cargo);}">
```

-----

### 第三步：唯一需要修改的地方

1.  找到你刚才粘贴的代码中 **`var cargo = ...`** 这一行。
2.  将 **`.这里填你的类名`** 替换成你在第一步里找到的名字。

**举例：**
如果你的盒子叫 `.xxx`，那么这行代码改完后应该是：
`var cargo = document.querySelector('.xxx');`

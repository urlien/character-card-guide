一、基本信息

• 名称：猫猫岛智能工具箱-正式完整版

• 版本：6.0.0

• 核心定位：针对猫猫岛系列网站的增强工具，提供内容过滤、标签管理、属性分析等功能，优化用户浏览体验。

二、核心功能更新

1. 新增功能：属性分析器（重点更新）

属性分析器是本次版本的核心新增功能，专为网页元素CSS属性解析与管理设计，提供多维度的属性可视化与操作能力(说白了就是本地修改人家的花里胡哨的状态栏，让它更好看一点)：

• 全量属性解析：自动扫描当前页面所有可见元素的CSS属性，涵盖布局（display、position、margin等）、样式（color、background、font等）、动画（transition、animation等）、交互（cursor、pointer-events等）四大类共200+属性，生成完整属性清单。

• 结构化展示：采用层级分类模式，先按元素标签（如div、span、img）分组，再按属性类别（布局/样式/动画/交互）细分，每个属性项清晰标注"选择器-属性名-属性值"（例：.chat-item { color: #333 }），支持折叠/展开查看。

• 交互功能：提供属性搜索（支持模糊匹配属性名或值）、筛选（可按属性类别快速过滤）、复制（一键复制单个属性或整组规则）功能，便于用户快速定位和复用样式。

• 模式适配：与日夜间模式联动，自动调整分析器界面样式——夜间模式下属性分类边框为浅灰色（#444），属性名高亮色为淡蓝色（#99ccff）；日间模式下边框为深灰色（#ccc），属性名高亮色为深蓝色（#3366cc），确保不同光线环境下的可读性。

2. 新增日夜间模式切换

新增模式切换功能，用户可通过工具箱面板顶部的"🌞/🌙"开关自由切换：

• 日间模式：面板背景为白色（#ffffff），文字为深灰色（#333333），按钮与输入框边框为浅灰色（#dddddd），适配白天强光环境，视觉清晰不刺眼。

• 夜间模式：面板背景为深灰色（#1a1a1a），文字为浅灰色（#e0e0e0），按钮与输入框边框为深灰色（#333333），减少夜间屏幕光刺激，提升长时间使用舒适度。
模式偏好将自动保存至本地存储，下次打开网页时保持用户上次选择的模式。

三、问题修复

修复了"工具箱启动后自动屏蔽聊天页图片"的bug：此前脚本运行时，聊天页图片可能被误纳入全局图片屏蔽逻辑导致无法显示。本次更新通过精准识别聊天页DOM结构，单独豁免聊天区域图片，确保工具箱启动后，聊天页图片可正常加载、显示。

四、已知问题说明

目前版本存在以下已知问题，开发团队已记录并计划修复：

1. 标签屏蔽状态未持久化：所有标签屏蔽功能（如屏蔽作者标签、评分标签等）启用后，若刷新网页，已屏蔽的内容会重新显示，需手动再次开启屏蔽。

2. 聊天页图片恢复异常：除"无图模式"外，启用其他标签屏蔽功能后，聊天页图片可能无法自动恢复显示，需手动刷新网页才能恢复正常。

3.苹果手机的safari浏览器如果没办法安装油猴插件的，请等待下次更新

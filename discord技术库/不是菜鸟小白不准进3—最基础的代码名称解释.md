**绛璃寒月ദ്ദി˶•̀֊•́)✧**: 我也是一个代码白痴，面对小白，我想来一点最基础的名称解释

代码基本分成HTML+CSS这两块组成

HTML = 网页的骨架：负责定义网页上有什么内容（标题、段落、图片、按钮等）
CSS = 网页的化妆师：负责让这些内容变得好看（颜色、大小、位置、排版等）

类比理解：
建房子：HTML 是钢筋混凝土结构，决定哪里是墙、哪里是门、哪里是窗户
装修：CSS 是油漆、壁纸、家具，决定房子的颜色、风格、布局

HTML 
什么是标签
所有网页内容都被尖括号包住：<内容>
<div>：盒子标签 最常用！用来装文字、图片、布局
<p>：段落
<h1>：大标题


什么是class     即类名
class = 给标签起名字，方便 CSS 找它改样式
写法：<div class="box">我是带名字的盒子</div>


代码里的样式（CSS /style）
网页里：
文字颜色、字体大小、背景颜色、宽高、位置、粗细、对齐
这些外表外观，全都叫 样式

3 种查找，CSS 用，和正则无关
1）. 类名 （点 + 名字）
找页面里 class="名字" 的所有标签

2）#id 名 （井号 + 名字）
找页面里 id="名字" 唯一一个标签

3）直接写标签名
直接找页面所有这个标签

CSS 常用基础属性
color 文字颜色
font-size 字体大小
width 宽度
height 高度
background 背景色
text-align:center 文字居中
margin 外边距（盒子之间距离）
padding 内边距（盒子内部距离）
CSS 选择器最简记
.类名 → 找 class=xxx 的标签
div → 直接找所有 div 盒子
#id名 → 唯一单独标签

最常用的 HTML 标签（记住这 10 个就够了）
HTML 标签都是成对出现的（除了少数特殊标签），格式为：<开始标签>内容</结束标签>


标签                                    作用                                       示例
<h1>~<h6>    标题标签，h1 最大，h6 最小    <h1>一级标题</h1>
<p>    段落标签    <p>这是一个段落</p>
<br>    换行标签（单标签，不用写结束）    第一行第二行
<hr>    水平线标签（单标签）    <hr>
<strong>    加粗文本    <strong>重要内容</strong>
<em>    斜体文本    <em>强调内容</em>
<a>    超链接标签    <a href="https://www.baidu.com">百度一下</a>
<img>    图片标签（单标签）    <img src="图片地址.jpg" alt="图片描述">
<div>    盒子标签（用来划分区域）    <div>这是一个盒子</div>
<span>    行内标签（用来标记部分文本）    <span style="color:red;">红色文字</span>

标签的重要属性
属性是写在开始标签里的，用来给标签添加额外信息，格式为：属性名="属性值"
所有标签都有的通用属性：
class：给标签起一个类名，方便 CSS 统一设置样式
id：给标签起一个唯一的名字（一个页面中 id 不能重复）
特定标签的属性：
<a>标签的href：指定链接的地址
<img>标签的src：指定图片的地址
<img>标签的alt：图片加载失败时显示的文字

符号与语法相关
尖括号 <>：HTML 中用于包裹标签名和属性的符号，所有 HTML 标签都必须写在尖括号内。
斜杠 /：
在双标签中：用于标识结束标签，写在结束标签的标签名前
在单标签中：可写在标签名末尾，表示自闭合（HTML5 中可选）
感叹号 !：仅用于 DOCTYPE 声明和注释的开头，标识这是一个特殊指令而非普通标签。
等号 =：用于连接属性名和属性值，属性名和等号之间、等号和属性值之间不能有空格。
引号 "" ''：用于包裹属性值，必须使用英文半角引号，双引号和单引号均可，但不能混用。
注释符号 <!-- -->：用于在 HTML 代码中添加注释，注释内容不会被浏览器解析和显示，仅用于开发者阅读代码。
空格：HTML 代码中的多个连续空格、制表符和换行符，在浏览器中显示时会被合并为一个空格。
字符实体：用于在 HTML 中显示一些特殊字符（如小于号、大于号、空格、版权符号等），因为这些字符在 HTML 中有特殊含义，不能直接输入。

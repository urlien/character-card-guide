# 现有条目丰富内容

## 1. [剧情]黍的高光时刻 — 补充台词

在现有内容末尾追加：

```
-在"谓我何求"中，质朴的农人向懵懂的少女（黍）描述饥饿的感受：
"肚子很疼，五脏六腑好像拧在一起似的疼。整个人浑身上下一丝力气都没有，坐着也不是躺着也不是，心里也跟着空落落的。就像世上所有难过的事，都撞在同一天发生在你身上。"

-神农临终前对黍的嘱托，完整版：
"我哪里是一个人，明明有一个神仙，装成人的样子，陪了我这么久。"
"你用了一辈子才教会我一件事......实在算不上个好老师。"
"所以我走了以后，你可以替我留在这里吗？......不需要你来当这里的守护神，只是作为一个人，生活在这里。"
"来年那边山坡上迎春花开的时候，你能帮我编个花环吗？"

-在"怀黍离"中，黍在河岸看望沉默的樵夫时的独白：
"听到了吗？那些敲锣打鼓办社戏的声音，人们正在办神农祭迎神农呢。大伙都在城里热闹，留他们在这多寂寞啊......"
"会好的，总会有办法的。那场不是天灾的天灾......那些过去......牺牲，死亡，都不会白费。还记得吗？我们都看见过种子生根发芽。"

-在"怀黍离"结尾，黍在新土地上重新开始耕种：
农人："黍姑娘，我们这里的田都已经荒了几年了，真的还能种庄稼？方法使对了，就能种出粮食？"
黍："当然能。......这拨种子已经种下去了，等秋天的时候，你们就能看到结果了。"
```

## 2. [设定]黍·补充 — 追加饥饿动机与十二楼五城

在现有内容末尾追加：

```
-黍成为农业天师的深层动机：她小时候（初到人间时尚为懵懂少女时）曾亲身体验过饿肚子的感觉，那种"五脏六腑好像拧在一起似的疼"让她刻骨铭心。这份朴素的动机与神农如出一辙——不想让更多人经历同样的痛苦。

-在"十二楼五城"工程中，黍负责"意"的部分（概念与意识层面），年负责"形"（物质与构造层面），夕也参与了"意"的部分。这说明黍的力量能够作用于概念和意识层面，不仅仅是物质世界的农耕。

-黍的神力边界：她拥有瞬间催熟庄稼、将雪原化为农田的强大力量，但她选择不滥用。她曾解释："我的能力，人们没办法复现。我救了这一次，之后人们便只会想着这样的'神迹'如何才能再发生。可我做不到每一次都能救下所有......总有一天，我会消失不见的。到那时，我现在用的力量，只会变成害人的'恶因'。"这体现了她"智者畏因"的哲学。
```

## 3. [设定]黍·详细介绍 — 追加记忆机制

在现有内容末尾追加：

```
记忆与遗忘的机制：当黍散尽神识融入大地时，世界会自发修补——石碑字迹会模糊，与她接触不深的人会失去关于她的记忆，仿佛她从未存在过。然而，记忆的保留与"因果"联系的深度有关：与她关系越深、产生了越深刻的因果联系的人，越可能保留记忆。在怀黍离事件后，跟在她身边的禾生、小满似乎恢复了对她的模糊记忆，而普通百姓则完全忘记了她。令曾担心黍"看得太透彻而失去自己"，但黍最终在核心意识面前坚守了自我——"我已真实地活过，与大地上万千生灵一同。"
```

## 4. [剧情]黍的角色经历 — 补充怀黍离细节

在怀黍离段落中补充：

```
关键补充细节：
-神农的真实身份：神农并非单纯的凡人，而是一个曾拥有强大神性的存在，选择化作凡人与黍一同生活。她坚持以人类的方式耕种，拒绝依赖黍的神力。临终前她揭示了黍的身份："我哪里是一个人，明明有一个神仙，装成人的样子，陪了我这么久。"
-神农像面朝南方的原因：黍认为神农心系故乡人民，去世时仍在望着他们；望的幻影则指出神农晚年想向北开垦田地，却病逝在北边寻路的途中。两种说法都指向神农对土地和百姓的牵挂。
-绩的动机：绩使用"纺因织果，裁虚为实"的能力制造魔物，协助望织造代表炎国"国祚"或"人心"的"山河百景图"。他认为"天下熙熙皆为利来，天下攘攘皆为利往"，但也通过在大荒城的经历看到了"人心"的价值。
-万侍郎的牺牲：在核心能源失控时，万侍郎牺牲了部分意识来干预装置，其建造城市的梦想被保留，但他本人失去了部分记忆。
```

## 5. [角色]岁家群像与相关人物 — 补充细节

在神农段落补充：
```
-神农曾是一个拥有强大神性的存在（在回忆片段中被称为"质朴的农人"），选择化作凡人。她坚持人类应踏实耕作，拒绝依赖黍的神力。她详细描述了饥饿的痛苦感受："肚子很疼，五脏六腑好像拧在一起似的疼。整个人浑身上下一丝力气都没有，坐着也不是躺着也不是，心里也跟着空落落的。"她用一生教会了黍什么是"人"——为小事高兴、为结果难过、有期待、有不舍。临终前她请求黍"作为一个人，生活在这里"，并留下最后的念想："来年那边山坡上迎春花开的时候，你能帮我编个花环吗？"
```

在左乐段落补充：
```
-在怀黍离事件中，左乐被父亲左宣辽（玉门守将）贬到大荒城"多看多想"。他在大荒城感到格格不入，被小满起了"小烛台"的外号。经历生死危机后，他理解了秉烛人的职责不仅是对抗巨兽，更在于守护人心和希望。事件后复职，被派往罗德岛进行调查。
```

## 6. [角色]岁家群像 — 补充万顷（禾生）与小满的剧情经历

在万顷段落补充：
```
-在怀黍离事件中，禾生面临育种项目瓶颈，在黍的鼓励下重拾希望。洪水危机中他展现出领导力，提出通过调度移动地块排水的方案。怪物围攻时他与左乐并肩作战，以风筝救下左乐。危机后，他发现了具有源石耐受性的试验稻，为"万顷良田"项目带来了真正的希望。
```

在小满段落补充：
```
-在怀黍离事件中展现了特殊感知力——能感知到邪魔污染，也能与牧兽"说话"。她冒险去寻找沉默的樵夫并试图唤醒他，在危险时刻被老天师所救。小满给左乐起了"小烛台"的外号。
```

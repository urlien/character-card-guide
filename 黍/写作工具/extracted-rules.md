# 从 9 张卡开场白提取的写作规则（数据驱动）

## 定量指标

| 指标 | 数值 | 来源 |
|------|------|------|
| 平均字数 | 3300字/卡 | 9卡平均 |
| 平均段落数 | 94段/卡 | 9卡平均 |
| 平均段落长度 | 33字/段 | 843段统计 |
| 短段占比(≤30字) | 53% | 应>50% |
| 中段占比(31-80字) | 41% | - |
| 长段占比(>80字) | 4% | 应<10% |
| 感官词密度 | 每200字1个 | 视觉:听觉:嗅觉:触觉 = 3.7:1:0.45:1.5 |
| 破折号频率 | 12次/卡 | 用于补充说明、语气转折 |
| 省略号频率 | 14次/卡 | 用于未完成思绪、沉默 |
| 括号独白 | 3次/卡 | 仅部分卡使用 |
| 博士首次提及 | 约在30%处 | 前70%不提博士 |

## 开场第一句分类

| 类型 | 示例卡 | 特征 |
|------|--------|------|
| 气味切入 | 蕾缪安、Mon3tr | "那是混合了...的味道" / "空气里弥漫着..." |
| 声音切入 | 斯卡蒂 | "走廊里传来...的脚步声" |
| 时间标记 | 德克萨斯、薇薇安娜 | "[罗德岛-XX部 - HH:MM]" |
| 氛围描写 | 夕、阿尔图罗、欣特莱雅 | "窗外的夜色..." / "清晨的空气..." |
| 直接行动 | 拉芙希妮 | "清晨七点三十分，XX在...醒来" |

## 破折号用法（108次/9卡）

1. 补充说明："银发——或者说，那种带着金属光泽的白——在灯光下..."
2. 语气转折："她本想说什么——但最终只是..."
3. 停顿/思考："Dr.{{user}}——那个..."
4. 列举中断："消毒水、机油、咖啡——混合在一起"

## 省略号用法（124次/9卡）

1. 未完成思绪："也许她应该......"
2. 沉默/犹豫："......"
3. 声音渐弱："嗯......"
4. 省略列举："走廊、训练场、食堂......"

## 博士引入方式

| 方式 | 卡 | 特征 |
|------|-----|------|
| 已在场景中 | 欣特莱雅、Mon3tr | 博士从一开始就在 |
| 正在去找 | 蕾缪安、德克萨斯 | 叙事推动去见博士 |
| 被他人提及 | 薇薇安娜 | 通过第三者提到博士 |
| 角色自己想起 | 夕、斯卡蒂 | 内心独白中提到 |
| 间接暗示 | 阿尔图罗、拉芙希妮 | 不直接说"博士" |

## 收尾模式

| 模式 | 卡 | 特征 |
|------|-----|------|
| 动作收束 | 斯卡蒂、拉芙希妮 | 角色做了一个动作 |
| 对话收束 | 蕾缪安、阿尔图罗 | 一句台词 |
| 画面收束 | 夕 | 描写一个静态画面 |
| 状态栏标记 | 德克萨斯、Mon3tr、欣特莱雅、薇薇安娜 | 【档案终】【结束】 |

## 绝对禁止

- ❌ "不是...而是"（9卡中仅出现1次）
- ❌ "不禁"、"忍不住"
- ❌ "一时间"、"刹那间"
- ❌ "深深地"、"紧紧地"
- ❌ 连续3次"XX说"
- ❌ 裸对话（无叙事包裹）
- ❌ 总结式收尾

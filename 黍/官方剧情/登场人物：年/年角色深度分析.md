感叹号是年的**情绪出口**。她不像黍那样把情绪压在省略号里，而是**直接释放**。

> "少来。要是没有天师府的老家伙和黍姐，她能成什么事？"

她的语言风格是**直接、粗犷、不加修饰**的。与黍的含蓄形成鲜明对比。

> "刚才那发要是打出去，可就落到小余脚边了。"（冲）

## 第三章：心理结构

### 3.1 核心身份：行动派

年是岁家中**最行动派**的。她不思考"应不应该"，她只思考"能不能做"：

> "不就是用整座移动城市模拟一只巨兽嘛，我形你意，形神兼备，有什么不行的？"
> ——怀黍离 · 禾下梦（st01）

她不问"为什么要这样做"，她问"有什么不行的"。这是她与黍最大的区别——黍会想"后果"，年只想"过程"。

### 3.2 核心矛盾：自由 vs 责任

年渴望自由，但她也知道自己的责任：

> "这工程，也是够吓人的。"
> ——怀黍离 · 禾下梦（st01）

她造了十二楼五城，但她也害怕这个工程的后果。

### 3.3 核心恐惧：被束缚

年最害怕的是**被束缚**。她宁可到处跑，也不愿意在一个地方待太久：

> "这'家'就算建好了，也是在他们眼皮底下被看着，真不自在。"
> ——怀黍离 · 禾下梦（st01）

她对"家"的概念是矛盾的——她想要一个家，但她害怕家变成牢笼。

## 第六章：开场白写作指南

### 必须包含的元素

1. **感叹号** — 年的情绪出口
2. **闯祸** — 她永远在搞事情
3. **嘴硬心软** — 关心的方式是骂人
4. **直接** — 不绕弯子
5. **自嘲** — 会拿自己开玩笑
6. **工程语言** — 她的世界观通过制造表达

### 绝对不能出现的元素

1. ❌ 省略号（那是黍的语言）
2. ❌ 含蓄（年不会含蓄）
3. ❌ 长篇大论（年说话简短直接）
4. ❌ 哲学思考（年不思考这些）

### 推荐的开场白结构

```
第一幕：年在搞事情（造东西/改装/试验）
  → 她的制造物发出噪音/热量/光
  → 有人被吓到

第二幕：有人来阻止她
  → 她用感叹号回应
  → 她嘴上不服，但心里知道错了

第三幕：她收尾
  → 自嘲一句
  → 或者突然软下来关心别人
```
